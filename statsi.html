Foundational Statistics

Your analytical comprehension of Psychological measurement can be (mostly) built on a single statistic: the correlation. There are several different correlation coefficients that are relevant for measurement, but your conceptual understanding is facilitated by a focus on only one coefficient: Pearson’s product moment correlation (aka “correlation”). In this chapter, we will walk you through the way Psychometricians think about this correlation coefficient—conceptual understanding of this index is facilitated by first understanding its ancestral grandparent, the deviation score.

Toward an index of “difference” within a set of numbers, we could find the difference between each number and every other number within a set of numbers. For example, \(2-1=??\), \(3-1=??\), \(4-1=??\), etc. This process gets very unwieldy very quickly even with a relatively small number of numbers.

The typical strategy here is to NOT use every individual value, but to use a summary of every individual value as a comparison against which differences can be evaluated. By convention, the summary that is used is the mean (aka mathematical average):

\[\bar{X} = \frac{\sum{X_i}}{n}\]
